# eDaemon Game Engine

## English
### What is eDamon?
The eDaemon is a game engine for digital games. It was based on Daemon, the first brazilian tabletop RPG system.
Even though eDaemon is based on a tabletop RPG system it is not meant just for RPG games. It can be used on a variety of game types like *Hack'n'Slash*, *Real Time Strategy* and much more.
This game engine is free - and it will always be free - for you to download, change, use it and any other thing - covered by MIT license - that you want to do with it. As a counterpart you agree on use eDaemon to create fantastic and fun games!

### What is Daemon tabletop RPG system?
Daemon system was published for the first time on 1992 by Marcelo Del Debbio on a fanzine. Later, on 1995, the system got known to the general audience through the ARKANUN RPG, published by Trama Editorial.
This system was used in many other tabletop RPG books (like *Invasăo* and *Trevas*), game suplements (like *Grimório*, a magic and spell compendium for Daemon games) and it even had a series of complete adventures launched. However this system had no official name. It was still known as "the Trevas System" or "the Arkanun System" until the Daemon Publisher bought it's rights and renamed it *Daemon System*.
It is a generic system that allows you to use it in a variety of RPG styles. It allows you to dyscover medieval high-fantasy realms, to tell stories of hiden horrors in a modern world and even cast the players on superheroic stories to wach them save the day!
Read more about Daemon system and many other books [on Daemon Publisher website] (https://www.daemon.com.br/) and contribute to make the brazilian RPG better!

## Portuguęs
### O que é o eDaemon?
O eDaemon é um motor básico para jogos digitais. Foi desenvolvido a partir das regras do Daemon, o primeiro sistema de RPG de mesa brasileiro.
Apesar do eDaemon ser baseado em um sistema de RPG de mesa, ele năo se limita a jogos de RPG eletrônico; ele também pode ser usado em jogos de *Hack'n'Slash*, *Real Time Strategy* e outros tipos diferentes de jogos digitais.
Éste sistema é gratuito - e sempre será - para vocę baixar, copiar, alterar, usar e qualquer coisa que vocę queira fazer, dentro das regras da licença da MIT. Em contrapartida vocę deverá usá-lo para criar jogos incríveis e divertidos!

### O que é o sistema de RPG Daemon?
O sistema Daemon foi publicado pela primeira vez em 1992 por Marcelo Del Debbio em uma fanzine. Mais tarde, em 1995, o sistema ficaria conhecido pelo público geral através do RPG ARKANUN, publicado pela Trama Editorial.
O Daemon foi utilizado em outros livros de RPG de mesa, como *Invasăo* e *Trevas*, em diversos suplementos, como o *Grimório*, um compęndio de magias para Daemon, e até foi feito uma série de aventuras completas. Porém, o sistema năo tinha um nome oficial, sendo conhecido como "o Sistema do Arkanun", ou "o Sistema do Trevas", ou "o sistema do Só Aventuras". Até que a Editora Daemon foi criada e comprou os direitos sobre o sistema batizando-o como *Sistema Daemon*.
É um sistema genérico, permitindo o seu uso em aventuras de RPG dos mais variados tipos. Suas regras permite desbravar mundos de fantasia medieval, narrar horrores escondidos em um mundo moderno e até lançar os jogadores em aventuras de superheróis evitando catástrofes mundiais!
Saiba mais sobre o sistema Daemon e outros livros [no site da Editora Daemon] (https://www.daemon.com.br/) e contribua para o RPG brasileiro!
